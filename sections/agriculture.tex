\section{Results: Agriculture}
\label{sec:agriculture}

This section presents diagnostic results and end-of-century projections for agricultural damage functions across major crop categories. All results are presented at the \textbf{Impact Region (IR) level}.

Each crop subsection includes:
\begin{itemize}
    \item Summary statistics (placeholder)
    \item Model fit diagnostics
    \item Slope and polynomial analysis
    \item Worst-performing impact regions
    \item Scenario-based projections (SSP $\times$ RCP $\times$ Model combinations)
\end{itemize}
